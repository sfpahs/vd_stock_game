# UI 레이아웃 (6탭 구성)

## ⚠️ 디자인 구현 원칙

**모든 UI 디자인은 반드시 `.ui` 파일(Qt Designer)로 구현한다.**

- 위젯 생성, 레이아웃, 속성 설정 등 시각적 요소는 코드에서 직접 작성하지 않는다.
- `.ui` 파일을 `uic`로 컴파일하여 `ui_*.h`를 생성하고 C++ 코드에서 참조한다.
- 예외 없음: 동적으로 생성해야 하는 위젯도 최대한 `.ui` 기반 구조를 활용한다.

## Tab 1: 포트폴리오

```
현금(KRW): 50,000,000원  |  현금(USD): $10,000.00
총 평가액: ---  |  총 손익: ---  |  💱 1,471
┌─────────────────────────────────────┐
│ 종목코드 │ 종목명 │ 수량 │ 단가 │ 현가 │
├─────────────────────────────────────┤
│ 005930   │ 삼성   │ 100  │70K  │78K  │ KRW
│ BTC      │ BTC    │ 0.5  │$35K │$41K │ USD
└─────────────────────────────────────┘
```

**위젯**:
- QLabel: KRW/USD 잔액, 평가액, 손익
- QTableView (PortfolioModel): 7컬럼
  - 통화 구분 색상 (KRW: 검정, USD: 파랑)

---

## Tab 2: 매매

```
종목: [005930 ▼]
현재가: 78,000원
수량: [100___]
총금액: 7,800,000원
[매수 빨강]  [매도 파랑]  [주문유형 ▼]
```

**위젯**:
- QComboBox: 종목 선택 (FX 제외)
- QLabel: 현재가
- QSpinBox: 수량
- QDoubleSpinBox: 주문가격
- QPushButton: 매수/매도
- QComboBox: MARKET/LIMIT 선택

**신호-슬롯**:
- 종목 변경 → 현재가 갱신
- 수량/가격 변경 → 총금액 계산
- 매수/매도 → buyStock/sellStock

## 상단 네비게이션

```
┌─ 수익률: +12.3%  ────────  환율: $1=₩1,471  ────────  [설정]┐
```

**위젯**:
- QLabel: 수익률 (색상: 양수 빨강 / 음수 파랑)
- Ticker: 환율 실시간 표시
- QPushButton: 설정 버튼

---

**주요 색상**:
- KRW: 검정 (원)
- USD: 파랑 ($)
- 수익: 빨강 (+)
- 손실: 파랑 (-)
