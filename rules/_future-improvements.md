## Priority 2: 사용성 개선 (Q3 2026)

### 2.1 Stop-Loss / Take-Profit
```sql
새로운 orderType:
- STOP_LOSS: 손실 제한 (가격 하락 시 매도)
- TAKE_PROFIT: 수익 고정 (가격 상승 시 매도)
- TRAILING_STOP: 최고가 기준 -5% 하락 시 매도
```

### 2.2 포트폴리오 리밸런싱
```
현재: 주식 80%, 코인 20%
목표: 주식 60%, 코인 40%
→ 자동 매도/매수로 리밸런싱
```

### 2.3 거래 수수료 & 세금
```
각 거래에 수수료 적용 (0.1~0.5%)
거래세 시뮬레이션 (증권거래세 0.23%)
실제 수익률 계산
```

### 2.4 포트폴리오 공유 & 백테스트
- 공유: PDF 내보내기, 스냅샷
- 백테스트: 과거 데이터로 전략 검증
